# Лабораторная работа №1 — GuitarStrings

## Описание

GuitarStrings — Android-приложение, имитирующее игру на гитаре с помощью датчика приближения. При поднесении пальца к датчику приложение воспроизводит звук выбранной струны, вибрирует и анимирует её на экране.

## Функции

* выбор активной струны нажатием;
* обнаружение жеста через датчик приближения;
* воспроизведение тона при срабатывании датчика;
* вибрация при каждом "щипке";
* визуальная анимация струны (вспышка);
* отображение названия текущей ноты.

## Технологии

* Java
* Android SDK
* XML
* SensorManager / Proximity Sensor
* ToneGenerator
* Vibrator

## Разрешения

```xml
<uses-permission android:name="android.permission.VIBRATE" />
```

## Инструкция по запуску

1. Клонировать репозиторий:
```bash
git clone https://github.com/Ja1k3r37/GuitarStrings.git
```
2. Открыть проект в Android Studio.
3. Подключить реальное устройство (эмулятор не поддерживает датчик приближения).
4. Нажать **Run**.

## Использование

1. Запустить приложение.
2. Нажать на одну из 6 струн для выбора ноты (E2, A2, D3, G3, B3, E4).
3. Поднести палец к датчику приближения — струна «зазвучит».
